Loading file

Subject: The World Wide File System

Actors involved: The user, the web browser, Application server, Database Requirements addressed:

  • As an user, I need to be able to load the file into my web browser for which I have access to.

    • Precondition: The user has logged in and opend the file which the user wants to load.

    Typical flow:

    1. The user requests the file server to load the file which he/she has opened

    2. The system loads the file onto the web browser where the user wishes to see the loaded file

    Postcondition: The file required by the used is loaded on to the web browser, and the user is still logged into the system
